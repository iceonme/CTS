# TradeMind - 产品愿景

> **让每个人都能拥有专业的 AI 交易团队**

**版本**: v1.0  
**最后更新**: 2026-02-18

---

## 一句话定位

**TradeMind** 是你的**交易智能体小队**——一个由 AI 组成的虚拟交易团队，7x24 小时为你盯盘、分析、执行交易。

---

## 我们要解决什么问题？

个人投资者在数字资产市场面临的困境：

| 困境 | 现状 |
|------|------|
| **没时间盯盘** | 市场 24/7 运转，人有工作睡觉 |
| **信息过载** | 新闻、推特、数据太多，不知信谁 |
| **情绪干扰** | 恐慌和贪婪导致追涨杀跌 |
| **缺乏专业工具** | 机构有量化团队，散户只有直觉 |
| **学习门槛高** | 技术分析、风险管理需要长期积累 |

## 我们的解决方案

### 核心概念：AI 交易智能体小队

不再是一个人战斗，而是拥有一支**专业 AI 团队**：

```
你（指挥官，做最终决策）
    ↓
PA (Personal Assistant) - 你的交易小队队长
    ├─ 理解你的风险偏好和投资目标
    ├─ 协调各专家 Agent 分析市场
    ├─ 给出平衡的建议（Bull vs Bear 双视角）
    └─ 执行你授权的交易
    
情报 Agent 网络（各领域专家）
    ├─ 📊 技术分析员：指标、图表、趋势
    ├─ 🎯 宏观分析员：政策、事件、周期
    ├─ 💰 情绪分析员：市场情绪、社交舆情
    └─ 🐋 巨鲸监控员：链上资金流动
```

### 你的小队成员

| 成员 | 职责 | 特点 |
|------|------|------|
| **👔 PA (小队队长)** | 团队协调、战术决策、与你对话 | 领导小队，用 Bull/Bear 双视角分析，给你最平衡的建议 |
| **📊 技术分析员** | 监控技术指标（RSI、MACD、均线等） | 只看数据，不猜测 |
| **🎯 宏观分析员** | 解读政策、事件、周期 | 关注基本面和大趋势 |
| **💰 情绪分析员** | 分析市场情绪、社交舆情 | 捕捉恐慌与贪婪 |
| **🐋 巨鲸监控员** | 追踪大户资金流向 | 发现庄家动向 |

### 核心能力

**1. 智能盯盘（主动服务）**

不是等你问，而是主动告诉你：

```
15:30 📱 手机推送

PA: "老板，BTC 出现突破信号！
     技术面的 RSI 刚上穿 70，
     宏观面显示 ETF 预期升温，
     建议关注。要买入 10% 仓位吗？"
```

**2. 多维度分析（集体智慧）**

不是单一观点，而是团队讨论：

```
用户："BTC 现在能买吗？"

PA: "让我召集团队分析一下..."

📊 技术分析员："RSI 62，趋势向上，突破 24h 高点"
🎯 宏观分析员："ETF 通过概率上升，市场乐观"
🐋 巨鲸监控员："检测到 500 BTC 流入交易所，警惕"

PA: "综合判断：技术面偏多，资金面偏空。
     建议小仓位试探（5%），设置止损。"
```

**3. 可进化的小队**

不是静态程序，而是会成长的助手：

```
Week 1: PA 建议买入 BTC，你没听，结果涨了 10%
       → PA 学习："用户偏保守，下次建议更明确"

Week 4: 技术分析员发现它预测的 RSI 超卖准确率 80%
       → 提高该 Agent 的信号权重

Week 12: 群体记忆积累 100+ 次交易经验
        → 形成 "BTC 减半前 3 个月规律" 等洞察
```

## 产品演进愿景

### Phase 1: 挑战赛验证（当前）
**目标**: 验证 MAS 架构有效性，发现最佳策略

让不同类型的交易者（人类基准、AI 单兵、MAS 小队）在**相同历史数据**上竞技，比较资产增值能力，找到最优策略组合。

### Phase 2: 个人助手（MVP）
**目标**: 服务个人投资者的基础版本

- 用户可配置自己的 Agent 小队
- 基础盯盘和提醒功能
- 模拟交易练习
- 简单对话交互

### Phase 3: 智能协作网络
**目标**: 集体智慧，Agent 间协作

- Agent 可以雇佣/解雇
- Agent 之间可以讨论和辩论
- 用户可以看到其他成功用户的 Agent 配置
- 策略市场（分享和购买策略）

### Phase 4: 自动化执行
**目标**: 真正的"自动驾驶"

- 接入交易所 API（用户授权）
- 分级授权（建议/半自动/全自动）
- 风控系统（止损、仓位管理）
- 合规适配

### 终极愿景：交易社交网络

从个人助手到社区生态：

```
TradeMind 交易社交网络
├─ 个人层：用户 + 个人 Agent 小队
├─ 分享层：Agent 创作者市场
│   ├─ 明星技术分析员 Agent（被 1000+ 人订阅）
│   ├─ 宏观大师 Agent（预测准确率 85%）
│   └─ 你的自定义 Agent（分享给朋友）
└─ 共识层：全网群体智慧
    ├─ 实时市场情绪指数（基于所有 Agent）
    ├─ 集体预测准确率（透明可验证）
    └─ 最佳策略排行榜
```

**关键创新**:
- **Agent 即服务**: 优秀的 Agent 可以被订阅，创作者获益
- **透明可验证**: 所有预测和交易都可追溯，无法造假
- **群体智能 > 个体**: 集体的智慧比任何单一专家都准

## 🏗️ 核心架构逻辑：三层驱动模型

为了实现“环境无关性”并最大化信息利用效率，TradeMind 遵循以下三层架构：

| 层次 | 名称 | 职责 | 核心组件 |
|------|------|------|------|
| **L1: 数据层** | **Data Layer** | 提供客观事实（K线、概率、指标原始数据） | DuckDB, Exchange SDK |
| **L2: 调用层** | **Calling Layer** | 逻辑加工与分析工具，供程序或 LLM 直接调用 | Skills, Function Calls |
| **L3: 信号层** | **Feed Layer** | MAS 协作空间，Agent 交换观点与辩论 | FeedBus, Agent Discussions |

### 驱动隔离模式
- **Arena (加速驱动)**: 外部泵（RaceController）通过注入 VirtualClock 加速历史数据流转。
- **Production (原生驱动)**: 内部心脏（Clock/Socket）实时获取并处理真实数据信号。

## 设计原则

1. **人机协作，而非替代**  
   AI 提供建议和分析，最终决策权在用户

2. **可解释，可验证**  
   每个建议都有明确理由，每个预测都可追溯验证

3. **渐进式自动化**  
   从建议 → 半自动 → 全自动，逐步建立信任

4. **开放生态**  
   Agent 可开发、可交易、可组合
