# CryptoPulse AI: UI 设计文档

> **版本**: v1.0 | **五大模块**: 首页(CFO对话) | Feed流 | WarRoom | 资产交易 | 账户配置

---

## 1. 整体布局架构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  🟦 顶部导航栏 (Navigation Bar)                                              │
│  Logo  [Feed] [WarRoom] [资产] [配置]    [搜索] [通知] [用户头像]           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ┌──────────────┐  ┌──────────────────────────────────┐  ┌──────────────┐  │
│  │              │  │                                  │  │              │  │
│  │   🟨 左侧    │  │         🟩 主内容区              │  │   🟧 右侧    │  │
│  │   边栏       │  │                                  │  │   边栏       │  │
│  │              │  │    (根据当前模块切换内容)         │  │              │  │
│  │  • 快速导航  │  │                                  │  │  • 市场概览  │  │
│  │  • 关注列表  │  │                                  │  │  • 热门话题  │  │
│  │  • MAS成员   │  │                                  │  │  • 推荐关注  │  │
│  │              │  │                                  │  │              │  │
│  └──────────────┘  └──────────────────────────────────┘  └──────────────┘  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

**布局规则**:
- 桌面端: 三栏布局 (左:250px | 中:自适应 | 右:300px)
- 平板端: 两栏布局 (左:80px图标 | 中:自适应)
- 移动端: 单栏 + 底部导航
