## 🎯 一句话定位

> **TradeMind** 是你的**交易智能体小队**——一个由 AI 组成的虚拟交易团队，7x24 小时为你盯盘、分析、执行交易。

---

## 🎭 面向谁？

**主要用户**: 非技术背景的数字资产投资者

**用户画像**:
- 想投资但没时间盯盘的白领
- 需要专业分析但不信任单一信号的散户
- 希望系统化交易的新手

**不需要**: 编程能力、AI 知识、复杂配置

## 🚀 核心功能

### 1. 智能盯盘（主动服务）

**不是等你问，而是主动告诉你**

```
15:30 📱 手机推送

PA: "老板，BTC 出现突破信号！
     技术面的 RSI 刚上穿 70，
     Polymarket 上 BTC 突破 7 万的概率也在上升，
     建议关注。要买入 10% 仓位吗？"
```

### 2. 多维度分析（集体智慧）

**不是单一观点，而是团队讨论**

```
用户："BTC 现在能买吗？"

PA: "让我召集团队分析一下..."

📊 技术分析员："RSI 62，趋势向上，突破 24h 高点"
🎯 Polymarket 专员："ETF 通过概率 78%，市场乐观"
🐋 巨鲸监控员："检测到 500 BTC 流入交易所，警惕"

PA: "综合判断：技术面偏多，资金面偏空。
     建议小仓位试探（5%），设置止损。"
```

### 3. 模拟交易（零风险学习）

**用虚拟资金练习，再实盘**

- 初始资金：10,000 USDT（虚拟）
- 真实价格：对接 CoinGecko 实时数据
- 完整记录：每笔交易的盈亏、胜率统计
- 策略验证：测试 PA 的建议是否有效

### 4. 可进化的小队（越用越懂你）

**不是静态程序，而是会成长的助手**

```
Week 1: PA 建议买入 BTC，你没听，结果涨了 10%
       → PA 学习："用户偏保守，下次建议更明确"

Week 4: 技术分析员发现它预测的 RSI 超卖准确率 80%
       → 提高该 Agent 的信号权重

Week 12: 群体记忆积累 100+ 次交易经验
        → 形成 "BTC 减半前 3 个月规律" 等洞察
```

## 🌟 产品差异化

### vs 传统交易机器人

| | 传统机器人 | TradeMind |
|--|-----------|-----------|
| **决策方式** | 单一策略（金叉买死叉卖）| 多 Agent 协作、交叉验证 |
| **适应性** | 固定规则 | 可学习、可进化 |
| **透明度** | 黑盒 | 每个决策都有理由 |
| **交互** | 配置参数 | 自然语言对话 |

### vs 单一 AI 助手

| | ChatGPT/Claude | TradeMind |
|--|----------------|-----------|
| **专业度** | 通用知识 | 交易专用 |
| **实时性** | 无实时数据 | 实时价格、实时 Feed |
| **持续性** | 一次性对话 | 7x24 小时盯盘 |
| **记忆** | 无长期记忆 | 三层记忆、越用越懂你 |

## 🗺️ 路线图

### Phase 1: 核心小队（MVP）
**时间**: 4-6 周  
**目标**: 验证核心概念

- PA + 技术分析员 + Polymarket 专员
- 模拟交易系统
- 基础记忆系统
- 盯盘/找机会/异动/报告 四种任务

### Phase 2: 完整小队
**时间**: 4-6 周  
**目标**: 提升分析维度

- 新增：巨鲸监控员、情绪分析员
- War Room 可视化
- 完善群体记忆
- 策略回测系统

### Phase 3: 交易社交网络 ⭐
**时间**: 3-6 个月  
**目标**: 社区化、规模化

- **Agent 市场**: 用户可创建/分享自定义 Agent
- **信号共享**: 优秀 Agent 的信号可被订阅
- **群体智慧**: 全网 Agent 的共识形成市场指标
- **社交网络协议**: 基于去中心化身份


- **群体智能 > 个体**: 集体的智慧比任何单一专家都准
